Linux用户协会轻量级(light)网络加速服务章程

version: 1.0.2

总则

USTC LUG网络服务总则

服务介绍

中国科学技术大学Linux用户协会（以下称LUG或“我们”）light服务是一项为LUG会员提供的非营利性的网络加速服务。

功能

  • 国际网络加速

支持协议

  • HTTP 1.1
  • HTTP 2.0 TLS

账号的取得

申请对象

  • 科大在读本科生、研究生、博士生
  • 科大在职博士后
  • 科大在职教工
  • 在LUG中曾经担任职务的会员

申请材料

  • 姓名
  • 学号（10位）/工号（5位）
  • 科大邮箱（没有科大邮箱？点我注册）
  • 手机号（仅用于紧急联系）

申请方式

使用校园网访问在线申请网站提交您的申请材料。我们将在3个工作日内予以答复。

温馨提示：提交虚假的申请材料，将永久失去申请资格，已开通的账号亦将被封停。

账号的有效时间

在校期间有效。

限制

资源限制

  • 带宽：未限制
  • 月流量：20GB

功能限制

  • 仅限访问tcp 80、443、5228端口

滥用

为了保障服务器安全稳定运行，本服务禁止以下行为：

  • 分享账号
  • 网络扫描、网络攻击或其他黑客行为
  • 运行网络爬虫
  • 可能导致IDC投诉的行为
  • 违反中国法律法规或科大校纪校规的行为
  • 管理员认定的其他滥用行为

一经发现，管理员将视情节发出警告、暂停账号或永久封停账号。

隐私

所有经过服务器请求将被机器记录，LUG承诺不会对外泄露这些数据。记录保存时间为90天。

例外：当接到上级主管部门要求时，我们有权在不经过您同意的情况下提交这些数据。
